# DanmuAI 2026-05-29

建议 Git tag：`v2026.05.29`

## 亮点

- **直播网页弹幕层**：OBS / 直播伴侣可用浏览器源采集透明背景弹幕（SSE），与 Qt Overlay 并行，控制台「运行概览 → 直播输出」一键复制地址与测试弹幕。
- **Provider 适配层**：多服务商 OpenAI 兼容差异收口至 `app/providers/`（含小米 MiMo 专用适配），请求与连通性探测行为更一致。
- **小米 MiMo-V2.5 目录**：`mimo-v2.5`、`mimo-v2.5-pro` 展示名与定价对齐；默认视觉模型仍为 `mimo-v2.5`。
- **公告体验**：侧栏未读按公告 `id` 持久化（`config.db`）；顶栏可显示最新公告「标题：正文」前 30 字，可单独关闭。
- **稳定性**：截图退避接入 `screenshot_interval_ms`；移除 `probe_runnable` 死代码；WebSocket 路由修复；可选单实例（二次启动激活已有窗口）。

## 用户可见变更

### 新增

- **直播输出**（运行概览）：`/live-overlay` 透明页、`/api/live-overlay/events` SSE、`POST …/test` 测试弹幕；AI 弹幕上屏时同步到网页层（视觉 + 开麦）。
- **Provider 适配层**：火山方舟 / 百炼 / 硅基流动等走默认 OpenAI 适配；小米 MiMo 走专用适配（`thinking` 等）。
- MiMo 模型目录：**MiMo-V2.5**（`mimo-v2.5`）、**MiMo-V2.5 Pro**（`mimo-v2.5-pro`，无识图，不进视觉下拉）。
- 公告：**已读状态 API**（`GET/PUT /api/announcements-read-state`）；顶栏简略条（`localStorage` 记录已关闭 id）。
- Web 静态资源：**内置 Tailwind**（`tailwindcdn.js`），减轻外网 CDN 不可用时的样式降级（见已知问题）。
- **单实例**：本机重复 `python main.py` 时激活已有控制台窗口并退出副本进程。

### 移除

- `app/probe_runnable.py`（库存预取探测，主链路未使用）。

### 变更

- 主链路：移除 realtime / 节奏模式遗留；连续 API 失败时截图间隔退避（`live_freshness` → `screenshot_interval_ms`）。
- MiMo：OpenAI 兼容请求与默认视觉模型对齐；README 服务商表更新。
- Web：`app/web_console.py` 使用 Starlette `WebSocketRoute`；日志 WS 异常码 1008 时刷新会话。
- 截图：拒绝 null / `isNull()` / 零尺寸 pixmap，不递增 `screenshot_id`。
- RTT 计时键：`{request_round}:{screenshot_id}:{scene_generation}`，避免麦克风与视觉请求互相覆盖。
- `ConfigStore.set()` 写入失败时不再污染内存缓存。

## 升级说明

1. **直播**：升级后打开「运行概览 → 直播输出」，将浏览器源 URL 设为显示的地址（默认 `http://127.0.0.1:18765/live-overlay`），OBS 更新后请**刷新**浏览器源以加载新 `live-overlay.js`。
2. **小米 MiMo**：请在助手设置选择「小米 MiMo」预设、填写 Key 并**保存**；视觉模型推荐 `mimo-v2.5`（`mimo-v2.5-pro` 不支持识图）。
3. **公告**：若曾用旧版 `localStorage` 时间戳判断未读，首次启动会迁移到 `read_ids`；顶栏简略条关闭状态独立存储。
4. **不要用** `--qt-ui` / `DANMU_WEB_CONSOLE=0`（将 `exit 2`）。
5. Windows exe：见 [PACKAGING_WINDOWS.md](../PACKAGING_WINDOWS.md)；构建脚本不变。

## 已知问题

- `live_freshness` 本地慢模型 fallback（`build_local_fallback_batch`）尚未接入 main 主链路。
- MiMo 连通性探测仍为文本 ping，不覆盖识图请求。
- Google Fonts 等仍可能依赖外网；Tailwind 已内置，字体离线时可能回退系统字体。
- Supabase 公告/反馈 REST 外网不可达时相关页报错，不影响核心弹幕功能。
- 截图区域框选已支持；部分历史文档仍写「仅全屏」—以 [WEB_CONSOLE.md](../WEB_CONSOLE.md) 为准。
