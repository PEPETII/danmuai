## 附录：实现细节（汇报用）

### 默认值来源

- 后端 [`app/config_defaults.py`](../../../app/config_defaults.py) 的 `CONFIG_DEFAULTS` + `export_web_config_defaults()`
- `api_endpoint` / `model` 由 doubao 预设与 `default_catalog_model_id("doubao")` 推导
- 前端 `loadConfigDefaults()` 缓存，禁止另写 `DEFAULT_SETTINGS` 常量

### 恢复范围

| 范围 | 字段 |
|------|------|
| api Tab | `api_endpoint`, `api_mode`, `screen_index`, `model`, `temperature`, `max_tokens`, `mic_mode_enabled`, `mic_window_sec`, `memory_mode`, `memory_window` |
| capture Tab | 无（toast 提示） |
| danmu Tab | `normal_recognition_interval_sec`, `normal_reply_count`, `danmu_speed`, `danmu_lines`, `font_size`, `danmu_max_chars`, `opacity`, `dedup_threshold`, `layout_mode`, `hotkey`, `eviction_mode`, `empty_accel` |
| rhythm Tab | `image_max_width`, `image_quality` |
| 全部 | 上述并集 |

### API Key 保护

- `GET /api/config/defaults` 不含 `api_key`
- 恢复前 snapshot 输入框值，恢复后写回；`collectFormData` 仍仅在非掩码时提交 key

### 关键注释位置

- `app/config_defaults.py` — `export_web_config_defaults` 文档串
- `app/application/config_service.py` — `RESTORABLE_CONFIG_KEYS`
- `web/static/app.js` — `SETTINGS_RESTORE_GROUPS`、`applySettingsDefaults`
