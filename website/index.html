你的隐私，我们认真对待

从设计之初就将隐私保护融入每一个环节

截图不落盘

截图在内存中压缩后直接发送给 AI，默认不会保存到硬盘，也不会将截图原文写入日志。

密钥加密存储

API Key 使用 Fernet 对称加密存储在本地配置库中，保护你的密钥安全。

仅监听本机

Web API 仅监听 127.0.0.1，写操作需要 Bearer token 验证，防止外部访问。

常见问题

遇到问题？这里可能有答案

为什么启动后没有弹幕？

常见原因：API Key 未配置、截图失败，或连续失败进入退避。请在 Web「助手设置」检查 API 配置，在「弹幕日记」查看错误日志。

程序会保存截图吗？

默认不会。截图在内存中压缩后发送给 AI 模型，不会保存到硬盘。程序只保存弹幕文本历史。

如何配置 API Key？

启动程序后，在 Web 控制台打开「助手设置」，填写 API Endpoint、API Key 和 Model。在「服务商预设」中选择平台可自动填入默认地址。点击「保存配置」后回到「温馨控制台」点击开始即可。

Max Tokens 设得很低会怎样？

固定 5 条弹幕需要完整的 JSON/列表输出，过低会导致截断或解析失败。程序有下限保护（不低于 512），并固定关闭思考模式以确保输出稳定。

小米 MiMo 报「AI 返回为空」？

请使用 OpenAI 兼容模式 + 预设 endpoint，模型优先选择 mimo-v2.5。应用已强制关闭思考模式；若仍为空，请检查 Key 权限、配额与模型 ID 是否已在控制台开通。

Web 控制台打不开？

请确认 127.0.0.1:18765 端口未被占用，可尝试使用 --web-browser 参数在系统浏览器中打开。Windows 需要 WebView2 Runtime（Win10/11 多数已预装）。
